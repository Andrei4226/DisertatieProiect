Accessing the server:

- cd server
- node server.js

Accessing the client:

- cd client
- npm start

Opening the project in Visual Studio Code and then installing the following packages:

In the client directory (cd client):
- npm install

In the server directory (cd server):
- npm install multer

In the main project:
- npm install @emailjs/browser --save
- npm install axios
- npm install express

accesare server:
cd server
node server.js

accesare client
cd client
npm start


Deschidem proiectul in Visual Studio Code apoi instalam urmatoarele pachete:
	In client(cd client):
	npm install

	In server(cd server):
	npm install multer

	In proiectul principal:
	npm install @emailjs/browser --save
	npm install axios
	npm install express
